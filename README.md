# BioTech Solutions Website

A modern bioscience and biotechnology website built with React, TypeScript, Tailwind CSS, and Lucide React icons.

## Tech Stack

- React 19
- TypeScript
- Tailwind CSS
- Lucide React (icons)
- Vite

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:5173](http://localhost:5173) in your browser

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
